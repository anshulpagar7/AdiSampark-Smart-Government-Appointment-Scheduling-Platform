# Shabri Smart Appointment System

A modern appointment management platform designed for Adivasi Vikas Bhavan, Government of Maharashtra.

## Objective

To streamline appointment scheduling between citizens and officers through a digital booking system accessible via QR code.

## User Roles

* Citizen
* Staff
* Madam / Officer

## Core Features

* Walk-in Appointments
* Future Appointment Booking
* Queue Management
* Appointment Status Tracking
* Holiday Management
* Slot Blocking
* Multi-language Support (English, Marathi, Hindi)
* WhatsApp Notifications
* SMS Notifications
* Google Calendar Integration
